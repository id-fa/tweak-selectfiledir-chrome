# tweak-selectfiledir-chrome

This tool allows you to modify the download and file selection default directory for a specific Google Chrome profile.

## Requirements

- Windows only
- Python 3.9+

## Usage (Python)

Double click:

```
run.bat
```

## Usage (Standalone EXE)

Download the executable from Releases and double click.

No Python installation required.

## Recovery Method

If the profile becomes corrupted:

1. Close Chrome
2. Delete the `Preferences` file
3. Rename `Preferences.bak` to `Preferences`
4. Restart Chrome

## Disclaimer

Use at your own risk. Backup is automatically created.
